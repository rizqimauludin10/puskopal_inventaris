# Siakopal - Sistem Informasi Aset & Kerja Sama

Pusat Koperasi Angkatan Laut (Puskopal) Koarmada II

## Tentang Proyek

**Siakopal** adalah aplikasi berbasis web yang dirancang untuk memudahkan pengelolaan aset dan kerja sama di lingkungan Pusat Koperasi Angkatan Laut (Puskopal) Koarmada II. Proyek ini bertujuan untuk meningkatkan efisiensi pencatatan, monitoring, dan pelaporan aset, serta mempermudah manajemen perjanjian kerja sama secara digital.

Proyek ini dikembangkan menggunakan teknologi modern seperti:

- **CodeIgniter 4** sebagai framework backend
- **Tailwind CSS & Bootstrap** untuk antarmuka responsif
- **MySQL** sebagai basis data utama

Masalah yang ingin diselesaikan:

- Kesulitan dalam pencatatan dan monitoring aset secara manual
- Risiko kehilangan data atau dokumentasi kerja sama
- Kurangnya sistem terpusat yang mudah diakses oleh pihak terkait

## Fitur

- Pencatatan dan pengelolaan aset lengkap
- Manajemen perjanjian kerja sama
- Dashboard laporan aset dan kerja sama
- Notifikasi dan pengingat untuk kontrak/kerja sama yang akan berakhir
- Pencarian dan filter data cepat dan mudah
